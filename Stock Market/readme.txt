In order to run our stock server/client program all you need to do is to:

1) Unzip the files in a prefered folder 
2) Open terminal and navigate to the prefered folder 
3) Type 'make' to compile source codes
4) In Server side - To run the program type in this format: ./stockserver <port>
    In Client side - To run the program type in this format: ./stockclient <server ip> <port>

* Make sure to run the server first!

5) ENJOY!!!
